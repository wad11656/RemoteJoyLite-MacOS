MacOS <a href="https://github.com/wad11656/RemoteJoyLite-MacOS/releases/latest">Release</a> branch of RemoteJoyLite with built-in **Record** w/ audio input supported.

## Audio Hardware Requirements:

### 1. A **USB External Sound Card** with a 3.5mm **Line-In** port

  - e.g. **<a href="https://www.amazon.com/dp/B0CNVY89PF">Cubilux CB5</a>**, generic **<a href="https://www.amazon.com/dp/B07QBZVNPW" target="_blank">USB 6 Channel Audio Sound Card with SPDIF</a>** (both tested)
